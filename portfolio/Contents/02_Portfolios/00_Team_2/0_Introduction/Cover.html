Implementation Language

C# - Mono JIT compiler version 3.2.8 - www.mono-project.com

Reason for Language Choice

The reason for choosing C# with the Mono compiler is primarily because of our eperience with Java, being a similar language in syntax, but allowing us to gain experience in a new language. The Mono compiler is cross-platform and runs on a virtual machine, similar to Java, allowing us to work on whatever machine we desired for the project. Additionally, we wanted to stay away from the alternatives because we already had experience with Java and C++ but also didn't want to deal with debugging C++ code for semester project. C# also allowed for flexibilty during the scanner implementation phase due to it's use of 'goto' statements, which don't exist in Java.
